## Замечания

- luminance - светлота цвета, которая состоит из двух параметров:
    - tint - оттенок (белый) - смешение цвета с белым, где 1 - полностью белый
    - shade - полутон (черный) - смешение цвета с черным, где 1 - полностью черный
